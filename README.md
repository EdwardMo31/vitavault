# VitalVault · 健康知识宝库

> 用知识修护身体。

最初的想法，是做一个关于"健康"的程序。

**它不是诊所，而是一座知识宝库。** 它不诊断、不开药、不替你看病。

- **它收集并分享关于人体健康的全部知识，** 让任何人都能在这里普及医学常识，主动地关心身体。
- **它面向每一个人，** 了解健康知识不应该有门槛。

**一句话定位：** 一个把人体健康知识系统化收集、并向所有人开放的知识平台。

---

### 为什么叫 VitalVault？

- **Vital** — 生命力 / 至关重要
- **Vault** — 珍藏宝物的宝库
- 合起来：*珍藏要紧之物的地方*——把健康知识当作宝藏来收集与守护。

## 功能特色

- **像素人身体地图** — 点击身体部位查看对应系统的健康知识
- **疾病预防专区** — 血压、血糖、癌症、骨质疏松等常见病的预防视角
- **环境与生态** — 空气、饮水、绿色生活方式
- **全部文章 + 搜索** — 支持关键词搜索、系统筛选、收藏过滤
- **收藏 / 已读** — 本地存储，无需注册，隐私优先
- **医学参考 R** — 常用缩写、器官对照、药物 INN、化验试管颜色，4 个标签页实时搜索

---

## 技术特点

- 纯 HTML + CSS + JS，**无框架、无依赖**
- 单文件应用，**双击本地可直接打开**，完全离线可用
- 数据文件 `articles_data.js` 独立，方便扩充内容
- 响应式设计，桌面和手机均可正常使用

## 内容策略

| 类别 | 说明 |
|------|------|
| 身体系统卡 `a*` | 营养、运动、睡眠、心理等正向健康常识 |
| 疾病预防卡 `d*` | 高血压、糖尿病、癌症等常见病的预防角度 |
| 环境生态卡 `e*` | 空气质量、饮水安全、绿色生活 |
| 人工精编 `u*` | 高质量选题，人工撰写 |

**调性铁律**：只讲早识别 / 风险因素 / 降风险 / 何时就医，不讲诊断 / 用药 / 治疗细节。
